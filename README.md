## DEMO Gatling

**DEMO Gatling** is a project for load tests. <br/>

#### Requirements

- gradle >= 4.7

#### Getting Started

Install the dependencies, go to the directory `./gatling` and execute;
```
gradle build
```

Run tests;

```
./gradlew gatlingRun
```
